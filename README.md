# uk-translit

Official Ukrainian transliteration and slug generation for TypeScript, built according to the current rules of the Cabinet of Ministers of Ukraine Resolution No. 55. Designed for headless CMS workflows, with context-aware transliteration, clean slug output, strong test coverage, and a modern release pipeline.

## Installation

```bash
bun add uk-translit
```

## Usage

```typescript
import { greet } from 'uk-translit';

console.log(greet('World')); // Hello, World!
```
